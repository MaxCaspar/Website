# MaxCasper.com Brief

## Goal
Build a lean, fast personal website for Max Casper that showcases small code-gen projects and lightweight writing.

## Target Users
- Curious visitors discovering projects
- Potential collaborators/employers scanning quickly
- Max as the content owner maintaining projects and posts

## Product Shape
- Fullscreen generative ASCII terrain background (ambient, organic, subtle)
- Minimal foreground panels for intro, project cards, and writing list
- Projects are image-first with concise metadata
- Writing is markdown-backed and intentionally lightweight

## Core Constraints
- Pure vanilla HTML/CSS/JS runtime (no framework runtime)
- Fast initial load and low CPU animation behavior
- Readability-first layering over animated background
- Mobile-first adaptations where hover is unavailable

## Non-Goals (v1)
- No full CMS/admin interface
- No heavy visual effects that compete with content
- No backend-dependent personalization features
